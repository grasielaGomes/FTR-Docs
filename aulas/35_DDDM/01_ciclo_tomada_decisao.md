### Ciclo de Tomada de Decisão com Dados: Passo a Passo Prático

#### 1. Coleta de Dados

- Identifique a origem dos dados: dados próprios, bases externas, bancos de dados, etc.
- Avalie qual é o melhor local para buscar as informações.
- Considere se os dados serão gerados pelo próprio trabalho ou provenientes de terceiros.

#### 2. Análise e Processamento dos Dados

- Realize a limpeza dos dados: elimine dados incompletos, com erros ou em branco.
- Modele os dados para garantir qualidade e relevância.
- Durante a limpeza/modelagem, já comece a identificar quais dados são importantes para a análise.
- Exemplo: analisar se a região do usuário é relevante para o objetivo.

#### 3. Definição das Perguntas de Negócio

- Antes de analisar, defina claramente quais perguntas você quer responder com os dados.
- Selecione os dados que realmente vão ajudar a responder essas perguntas.
- O foco deve ser sempre nas perguntas que orientam a análise.

#### 4. Interpretação dos Dados

- Busque as respostas para as perguntas definidas.
- Interprete os resultados, mesmo que não sejam os esperados.
- O objetivo é entender a história que os dados contam, sem enviesamento.

#### 5. Tomada de Decisão

- Com base na interpretação, defina as ações estratégicas.
- A decisão pode ser mudar de caminho, implementar algo novo, manter o que está funcionando ou não fazer nada.
- O importante é que a decisão seja orientada pelos dados e pelo contexto analisado.

#### 6. Storytelling e Comunicação

- Utilize o storytelling para comunicar o processo: da coleta à decisão.
- Explique o porquê das perguntas, das respostas e das ações tomadas.
- O ciclo deve ser claro para todos os envolvidos no processo.

---

### Resumo do Ciclo

1. Colete os dados relevantes.
2. Faça o processamento e análise (limpeza/modelagem).
3. Defina as perguntas de negócio.
4. Interprete os dados e busque respostas.
5. Tome decisões estratégicas baseadas nos dados.
6. Comunique o processo e os resultados de forma clara.

O foco é sempre garantir que cada etapa contribua para uma decisão mais informada e alinhada com os objetivos do negócio.
