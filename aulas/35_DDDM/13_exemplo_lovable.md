### Exemplo Prático: Lovable AI e Técnicas Avançadas de Prompt Engineering

#### 1. Prompt Original Simplificado

- Solicitação: Crie um script em Python que leia um arquivo JSON com registros de vendas, filtre entradas acima de mil dólares e gere um gráfico em PNG.
- Output esperado: Imagem PNG mostrando as vendas filtradas.

#### 2. Prompt Avançado Estruturado

- Definição de papel: "Você é um engenheiro Python focado em automação de dados."
- Solicitação detalhada: Crie um script em Python bem documentado que leia um arquivo JSON chamado vendas.json, filtre registros com valor acima de mil dólares, gere um gráfico e um PNG mostrando as vendas filtradas.
- Comentários: Use comentários explicativos em cada etapa.
- Código: Escreva o código de forma clara e eficiente.
- Setorização: O prompt é construído por etapas, facilitando alterações e implementações em partes específicas do código.

#### 3. Importância da Setorização no Lovable

- Lovable AI responde melhor a prompts divididos em etapas ou setores.
- Permite alterar, implementar ou remover etapas específicas sem modificar todo o escopo.
- Facilita a obtenção de respostas assertivas, pois segue uma ordem lógica de execução.
- A ordem das etapas influencia diretamente o resultado esperado.

#### 4. Analogia com Receitas

- O desenvolvimento do prompt deve seguir uma lógica sequencial, como uma receita de bolo.
- Não é possível executar etapas finais antes das iniciais (ex: não se tira o bolo pronto do forno antes de preparar a massa).
- Pensar em etapas ajuda a IA a entender e executar corretamente cada parte do processo.

#### 5. Técnicas Avançadas Utilizadas

- **Systematic Role Framing**: Definição clara do papel da IA (engenheiro Python focado em automação de dados).
- **Progressive Deepening**: Setorização do prompt em etapas sequenciais.
- **Constraint Driven Prompt**: Restrição clara de output (gerar gráfico em PNG).
- **Output as Artifact**: Solicitação de um arquivo PNG como resultado final.

#### 6. Considerações Finais

- A ordem e a clareza das etapas são fundamentais para obter resultados assertivos no Lovable.
- O prompt estruturado facilita ajustes e garante que a IA compreenda exatamente o que se espera.
- A delimitação do output como PNG é tanto uma restrição quanto uma especificação do artefato esperado.
