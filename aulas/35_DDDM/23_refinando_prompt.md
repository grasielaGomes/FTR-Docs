### Refinando Prompts: Ajustes e Recomendações Práticas

#### 1. Detalhes Importantes sobre os Dados

- Os dados da planilha referem-se somente a um mês de informações sobre churn dos clientes.
- Por esse motivo, o Prompt 3 (análise temporal) não está adequado e não será utilizado.

#### 2. Recomendações para ChatGPT

- Informar claramente a limitação temporal dos dados.
- Adaptar os prompts para evitar análises temporais inviáveis.
- Incluir recomendações gerais ao final para evitar que detalhes sejam aplicados apenas a uma parte do processo.

#### 3. Recomendações para Manus AI

- Os prompts atuais estão complexos, especialmente os de modelagem preditiva.
- Não realizar modelagem preditiva.
- Prompts 2 e 3 estão complexos demais e devem ser simplificados.
- Direcionar restrições específicas, conforme discutido na aula de Prompt Engineering.

#### 4. Recomendações para Lovable

- Lovable é uma IA ótima para prototipação e criação de dashboards.
- Foco em dashboards e apresentação de storytelling, evitando direcionamento para criação de websites.
- Destacar que o objetivo principal é criar dashboards para análise dos dados, não websites.

#### 5. Dicas Gerais para Melhoria dos Prompts

- Melhore os prompts para que façam análises realmente relevantes, mas sem muita complexidade.
- Evitar prompts excessivamente complexos.
- Manus AI é indicada para relatórios, gráficos e extração de dados quantitativos.
- Lovable é indicada para dashboards interativos e trabalhos visuais.
- O foco deve ser a análise da planilha, priorizando o conteúdo e, em segundo plano, o aspecto visual.
