# Projeto Hands-on: Agente Secretária de IA

## 1. Objetivo do Projeto

- Criar um agente de IA com funções de secretaria.
- O agente terá ferramentas para agendamento e comunicação.

## 2. Ferramentas de Agendamento

- Função para consultar a data atual.
- Função para agendar um novo compromisso.
- Função para consultar compromissos do calendário.
- Função para remarcar um compromisso existente.

## 3. Ferramentas de Comunicação

- Função para consultar a caixa de e-mails.
- Função para enviar um novo e-mail.

## 4. Funcionamento do Agente

- O agente escolhe automaticamente qual função usar conforme o pedido do usuário.
- Exemplo: ao pedir para agendar um compromisso, o agente usa a função de agendamento.
- Exemplo: ao pedir para consultar a agenda, o agente consulta a data e depois o calendário.

## 5. Próximos Passos

- Implementar as ferramentas no código.
- Criar a chamada para a API do Gemini para integrar o agente.
