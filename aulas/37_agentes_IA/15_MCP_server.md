# Model Context Protocol (MCP) para Agentes de IA

## 1. O que é MCP

- MCP (Model Context Protocol) é um protocolo open source criado pela Anthropic.
- Serve para conectar IAs a sistemas externos, como ferramentas, recursos e dados.
- Define um sistema de regras para comunicação entre IA e servidores externos.

## 2. Conceito de Protocolo

- Protocolo é um conjunto de regras que define como agentes se comunicam.
- Exemplos de protocolos em situações cotidianas (jantar, caixa de supermercado).
- Seguir o protocolo garante comunicação eficiente e previsível.

## 3. Funcionamento do MCP

- MCP padroniza a comunicação entre IA e servidores/ferramentas externas.
- Permite que qualquer IA acesse qualquer recurso que siga o protocolo MCP.
- Facilita integração e aumenta o potencial de uso das ferramentas por IAs.

## 4. Vantagens do MCP

- Elimina necessidade de criar código específico para cada integração.
- Empresas podem expor seus recursos via MCP Server, tornando-os acessíveis para qualquer IA.
- IA se torna mais poderosa ao acessar múltiplos recursos padronizados.

## 5. Exemplos de Uso do MCP

- Servidores de e-mail, agenda, ferramentas locais, prompts e dados podem ser expostos via MCP.
- IA consulta servidores MCP para acessar funções, prompts ou dados sem precisar de integração manual.

## 6. Catálogo de Ferramentas e Recursos

- MCP Server funciona como um catálogo de funções, prompts e recursos.
- Inclui metadados, descrições, inputs e outputs de cada ferramenta.

## 7. Impacto e Inovação

- MCP é recente e está sendo adotado por empresas e plataformas de IA.
- Abre novas possibilidades para agentes de IA, facilitando acesso a múltiplos recursos.
- Reduz complexidade de integração e acelera desenvolvimento de soluções inteligentes.
