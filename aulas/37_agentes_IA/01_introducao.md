# Introdução ao Módulo de Agentes de IA

## 1. Apresentação do Módulo

- Mascote do módulo: robô criado pelo Gemini, representando um agente de IA.
- Objetivo: entender o conceito de agentes de IA e aprender a criar e utilizar agentes.

## 2. O que são Agentes de IA?

- Explicação teórica sobre agentes de IA para quem não está familiarizado.
- Diferença entre IA generativa/LLM e agentes de IA.
- Importância de entender o funcionamento dos agentes antes de codificar.

## 3. Paradigma React (Reasoning + Acting)

- Não é o React do JavaScript, mas sim um novo paradigma para agentes de IA.
- Surgiu há cerca de 1-2 anos e revolucionou o desenvolvimento de agentes.
- Permite que agentes sejam mais inteligentes e capazes de agir de forma autônoma.

## 4. Uso de Ferramentas por Agentes

- Conceito fundamental para o funcionamento dos agentes.
- O uso de ferramentas é o que diferencia agentes de IA de modelos generativos tradicionais.
- Compreender React e ferramentas é essencial para entender agentes.

## 5. Mão na Massa: Criando um Agente

- Proposta de criar um agente "secretária de IA" capaz de acessar agenda, e-mails, etc.
- Estruturação do agente para integração com APIs externas (Google Calendar, Gmail).
- Foco em construir a base para agentes funcionais.

## 6. Model Context Protocol (MCP)

- Introdução ao MCP, protocolo recente e revolucionário para agentes de IA.
- Origem: proposto pela Anthropic, adotado por Google, Meta, OpenAI e outras empresas.
- Potencial para transformar o funcionamento dos agentes.
- Explicação sobre como criar e conectar um MCP Server.

## 7. Sistemas Multi-Agents

- Conceito de sistemas com múltiplos agentes trabalhando juntos.
- Área emergente com grande potencial de inovação.
- Desafio principal: implementação exige mais trabalho por envolver vários agentes.

## 8. Multi-Agents com LangGraph

- Utilização do LangGraph para facilitar a criação de sistemas multi-agents.
- Permite implementar agentes colaborativos de forma mais simples.

## 9. Resumo do Módulo

- Ao final, o aluno saberá:
  - O que é um agente de IA.
  - Por que agentes funcionam bem e como utilizam ferramentas.
  - Como codar agentes de IA e implementar ferramentas.
  - O que é MCP e como criar um MCP Server.
  - Como criar sistemas multi-agents e usar LangGraph para facilitar o processo.

---

O foco do módulo é fornecer a base teórica e prática para que o aluno compreenda, implemente e utilize agentes de IA em diferentes contextos.
